EDR

Enhanced Drash Resources

E' un framework per la realizzazione di microservizi e siti web compatibili con Deno Deploy.

E' dotato di una serie di funzionalità di base che possono essere utili in molteplici situazioni.
Questo sito viene utilizzato per sviluppare e testare le seguenti funzionalità:

  • Template engine per creazione di HTML dinamico lato server
  • Gestione di asset tramite CDN (Bunny.net)
  • Predisposizione per gestione pagine multilingua
  • Risorse JSON per gestione Rest API
  • Gestione registro chiamate in memoria integrato
  • Invio messaggi email tramite Web API (Resend)
  • Persistenza dei dati su Database MongoDB
  • Gestione telemetria su MongoDB
  • Monitoraggio servizi di sistema tramite logger dedicati
  • Gestione della autenticazione utente mediante OTP e JWT
  • Gestione utenti e relativa getione dei permessi di accesso alle risorse
  • Gestione invio email di richieste di supporto
  • Esecuzione interattiva degli unit test e analisi dei risultati
  • Gestione delle user story, del backlog e delle attività di sviluppo
