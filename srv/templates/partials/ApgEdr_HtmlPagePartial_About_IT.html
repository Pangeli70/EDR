Funzionalità del servizio EDR

Enhanced Drash Resources E' un framework per la realizzazione di microservizi compatibili con Deno Deploy. E' dotato di una serie di funzionalità di base che possono essere utili in molteplici situazoioni Le funzionalità di base sono:
  • Template engine per creazione server side di HTML dinamico
  • Rest API per invio di documenti JSON
  • Gestione log attività in memoria integrato
  • Interfacciamento ad una Web API (Resend) per l'invio di messaggi EMAIL
  • Interfacciamento a un Database MongoDB
  • Gestione telemetria su MongoDB
  • Gestione della autenticazione utente mediante OTP e JWT
  • Gestione utenti e relativa getione dei permessi
  • Gestione invio email di richieste di supporto
  • Esecuzione interattiva degli unit test e analisi dei risultati
  • Gestione delle user story, del backlog e delle attività di sviluppo
