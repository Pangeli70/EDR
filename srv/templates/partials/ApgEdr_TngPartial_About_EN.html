EDR

Enhanced Drash Resources

It is a framework for creating microservices and web sites compatible with Deno Deploy.

It is equipped with a series of basic features that can be useful in many situations.
This website is used to develop and test the following features:

  • Template engine for server-side creation of dynamic HTML
  • Assets management via CDN (Bunny.net)
  • Predisposition for management of multilingual pages
  • JSON resources for Rest API management
  • Management of call logs in memory integrated
  • Send email messages through Web API (Resend)
  • Data persistance on Database MongoDB
  • Telemetry on MongoDB
  • System services monitoring through dedicated loggers
  • Authentication user management through OTP and JWT
  • User management and management of access permissions to resources
  • Management of emails for support requests
  • Interactive execution of unit tests and analysis of results
  • Management of user stories, backlog and development activities
