Common Deno commands

Get current version: deno --version

Update to last version: deno upgrade

Common tasks (look at deno.json):
- run microservice: deno task run
- update local cache: deno task cacheLib deno task cacheSrv
- run the test suite (if available): deno task test

Technical debt management

When some portions of the code are identified as problematic use the following prefixes for the comments:

  • TODO: Activity to be completed
  • FIXME: Diagnosed error to fix
  • DEBUG: Potential error to diagnose
  • UGLY: Bad or smelly or poorly written code
  • HACK: Non orthodox 'trick' or an Undefined Behaviour that needs to be removed
  • REVIEW: Needs review
  • OPTIMIZE: Slow code that needs optimization
  • NOTE: Generic annotation
  • IDEA: Possible implementation of new features or algorithms

Add another line with a double hyphen '--', the author's acronym and the date in the universal format.

For instance:

// FIXME Extract the following block as reusable method
// -- APG 20220902
